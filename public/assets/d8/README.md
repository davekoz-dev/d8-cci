# D-8 Indonesia Chairmanship Assets

This directory contains brand assets for the D-8 Indonesia Chairmanship 2026-2027 website.

## Directory Structure

```
assets/d8/
├── logo/              # D-8 logo configurations (main, secondary)
├── supergraphics/     # SVG supergraphic variations
├── patterns/          # SVG pattern files at 40% opacity
├── photos/            # Curated photography
└── icons/             # Custom icons for priorities and features
```

## Logo Files

**Current Status:** Placeholder SVG files
**Required:** Replace with official D-8 Indonesia Chairmanship logo files

### Logo Configurations

1. **Main Configuration** (`d8-indonesia-logo-main.svg`)
   - Primary use throughout the website
   - Recommended format: SVG or high-resolution PNG
   - Minimum clear space: 2x logo height on all sides

2. **Secondary Configuration** (`d8-indonesia-logo-secondary.svg`)
   - For co-branding contexts with partner logos
   - Same technical requirements as main configuration

### Logo Usage Guidelines

- **Do NOT:**
  - Rotate, outline, or add effects to the logo
  - Place on complex or similar-color backgrounds
  - Recolor or modify the logo colors
  - Reduce size below minimum legibility threshold

- **DO:**
  - Maintain minimum clear space around logo
  - Use on approved background colors only
  - Ensure sufficient contrast with background
  - Use appropriate configuration for context

## Supergraphics

**Purpose:** Decorative background patterns following D-8 brand guidelines

Create SVG files with geometric patterns in approved color ways:
- White on Blue (Primary Blue background)
- White on Turquoise (Turquoise background)
- Blue on White (Soft White background)
- Turquoise on White (White background)

## Patterns

**Purpose:** Edge decorations at 40% opacity

Create SVG patterns for:
- Top edge decorations
- Bottom edge decorations
- Corner elements
- Side accents

All patterns must use only D-8 brand colors at 40% opacity.

## Photos

**Requirements:**
- Reflect Indonesian culture, cooperation, solidarity, prosperity
- High resolution (minimum 1920px width for full-width images)
- Formats: JPG, PNG, WebP
- Full-color, duotone, or supergraphic-framed variants acceptable

**Subjects:**
- D-8 member country cultures
- Economic cooperation scenes
- Summit and meeting imagery
- Indonesian landmarks and culture
- People-focused: 2-column layouts with clear legibility

## Icons

**Purpose:** Custom icons for priority areas and features

Create simple, geometric icons for:
1. Economic & Trade Integration
2. Halal Economy
3. Blue Economy & Green Transition
4. Connectivity & Digital Transformation
5. Organizational Reform

**Specifications:**
- SVG format
- 24x24px base size
- Single color (brand blue #055090 or turquoise #00B3AA)
- Stroke width: 2px
- Lucide React icons currently used as placeholders

## Image Optimization

Before adding images to this directory:
1. Optimize file sizes (use tools like ImageOptim, Squoosh, or Sharp)
2. Generate multiple sizes for responsive images
3. Use Next.js Image component for automatic optimization
4. Provide descriptive alt text for accessibility

## Brand Compliance

All assets in this directory must comply with D-8 Indonesia Chairmanship brand guidelines:
- Use only approved brand colors
- Follow logo usage rules
- Maintain consistent visual style
- Prioritize accessibility and legibility

For official brand assets, contact the D-8 Indonesia Secretariat.
